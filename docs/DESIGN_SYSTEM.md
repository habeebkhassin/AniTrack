# DESIGN_SYSTEM

## Theme

Background: #0D1117 Cards: #161B22 Borders: #2A313C Primary: #F2F5F7
Secondary: #98A2B3 Accent: #5FA8FF Success: #49D17D Warning: #FFB547
Error: #E65C5C

## Typography

Headings: Space Grotesk Body: Inter

## Icons

Primary: Lucide Fallback: Phosphor

## Card Rules

-   Rounded corners (small radius)
-   Thin borders
-   Subtle shadow
-   Consistent padding
-   Hover elevation

## Components

-   Poster Card
-   Continue Card
-   Progress Bar
-   Calendar Widget
-   Wisdom Widget
-   Journal Card
-   Statistic Tile
-   Character Card

## Motion

-   Fade: 150--250 ms
-   Scale: subtle
-   Slide: only for navigation
-   No flashy effects

## Spacing

Use an 8px spacing system (8/16/24/32).

## UX Principles

-   Information first
-   Numerical progress always visible
-   Calm, desktop-like experience
-   Minimal visual noise
